SD2BBN tool

SD2BBN tool for mapping interaction failure dependencies in UML Sequence Diagrams (UMLSD) to a reliability model in a Bayesian belief network (BBN)

The tool expects an UML 2.4.1. (XMI 2.4.2, http://www.omg.org/spec/UML/20110701) XML export of an Enterprise Architect (EA) Sequence Diagram.
